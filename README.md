# Bjerknes Diagnostic Lab: ENSO Dynamics

## Overview
This repository provides a computational exploration of the **Bjerknes Mechanism**, the fundamental feedback loop driving the El Niño-Southern Oscillation (ENSO). This project is designed as both a technical tutorial for students and a research-grade diagnostic tool for climate analysis.

## Key Objectives
- **Scientific Foundation:** Visualize the interaction between equatorial pressure gradients and atmospheric tendencies.
- **Diagnostic Tool:** Implement numerical diagnostics (zero isallobars and gradient maxima) to identify shifts in the Walker Circulation.
- **Experimental Climate Scenarios:** Simulate how basin-wide pressure anomalies ($\Delta\bar{p}$) trigger different ENSO regimes (Cold/La Niña vs. Warm/El Niño).

## Contents
- `Bjerknes-Diagnostic-Lab.ipynb`: The primary interactive notebook containing the narrative, math, and visualizations.
- `bjerknes_physics.py`: Modular Python code containing the underlying physics and diagnostic functions.

## Getting Started
To run this project, ensure you have Python 3.x installed along with the following libraries:
`numpy`, `matplotlib`, `pandas`.
